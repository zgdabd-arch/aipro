# **App Name**: ProAi

## Core Features:

- Student Profile Creation: Collect student's name, age, grade level, country, and preferred learning language to personalize the learning experience.
- Subject Selection: Allow students to choose the subject they want to improve in.
- Personalized Study Plan Generation: Generative AI builds a customized learning plan based on the student's curriculum, educational materials, and chosen subject.
- Progress Tracking: Track the student's progress and display it in a user-friendly format.
- Account Management: Enable users to sign up and log in with their email and password.
- AI Tutoring Tool: Use generative AI to provide personalized explanations and examples to aid student comprehension. This tool determines if and when this support should be incorporated into the session.
- Email Verification and Password Reset: Implement email verification during registration and password reset functionality for forgotten passwords.
- Subscription Model: Offer a paid subscription option to cover API and Firebase costs.

## Style Guidelines:

- Primary color: Soft sky blue (#87CEEB) for a calm, learning-focused environment.
- Background color: Light gray (#F0F0F0) for a clean, distraction-free backdrop.
- Accent color: Gentle green (#90EE90) for positive reinforcement and important callouts.
- Body and headline font: 'PT Sans' (sans-serif) for clear and accessible text.
- Use clear, simple icons related to learning and academic subjects.
- Clean, organized layout with a collapsible side navigation panel for key information.
- Subtle animations to indicate progress and engagement.
